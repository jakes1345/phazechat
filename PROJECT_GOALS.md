# Project: Xat.com Clone

## Objective
Build a fully customizable **Private Server Platform** compatible with the xat.com protocol, including:
- Retro chat interface with **Custom Asset Support** (smileys, sounds, backgrounds).
- Independent Trade Engine and **Local Economy Control** (minting xats/days in SQLite).
- Custom Power System (ability to add new bitmask-based powers).
- API and backend routes serving **local database content** instead of xat.com.
